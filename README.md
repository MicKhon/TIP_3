# Тренажёр по сложению

Простой интерактивный тренажёр для отработки навыка сложения двузначных чисел.

## Как запустить

1. Скачай файл `Primeri_primerov.html`
2. Открой его в любом браузере (Chrome, Firefox, Edge и т.д.)
3. Никакой установки или сервера не нужно — всё работает сразу!

## Как пользоваться

- На экране появляется пример вида `37 + 54 = ?`
- Вводишь ответ в поле и нажимаешь **«Проверить»** или **Enter**
- Если ответ правильный — появляется «Правильно!» и сразу новый пример
- Если ошибка — показывается правильный ответ
- Внизу ведётся статистика: правильные ответы, точность и лучший результат (сохраняется в браузере)

## Технологии

- **React 18** (через CDN, без сборки)
- Написано **без JSX** — только `React.createElement`
- Хранение лучшего результата — **localStorage**
- Один HTML-файл, без внешних зависимостей
